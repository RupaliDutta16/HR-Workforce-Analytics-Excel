# HR Workforce Analytics — Excel Dashboard

## Project Overview
Excel-based HR analytics dashboard analyzing workforce data including salary distribution, performance levels, promotion eligibility and department-wise insights.

## Dataset Details
- Employee Name, Age, Salary, Gender
- Department, Position, Joining Date
- Performance Score, PIP Status
- Promotion Eligibility, Years of Experience

## Sheets Structure
- **Data** — Cleaned and transformed raw data
- **Pivot_Table** — Analysis pivot tables
- **Insights** — Key findings
- **Dashboard** — Visual KPI dashboard

## Key Excel Skills Used
- Data Cleaning & Transformation
- IF, AND, OR, ROUND, TODAY functions
- Pivot Tables & Pivot Charts
- GETPIVOTDATA function
- Salary categorization (Low/Medium/High)
- Performance level classification
- PIP Status automation
- Promotion Eligibility logic

## Key Columns Created
- Salary Category → Low/Medium/High
- Performance Level → High/Medium/Low
- PIP Status → based on performance
- Promotion Eligibility → based on performance + experience
- Years of Experience → calculated from joining date

## Tools Used
- Microsoft Excel
- Pivot Tables
- Excel Formulas & Functions

## Key Insights
- Department-wise salary performance analysis
- Promotion eligibility by department
- Gender distribution across departments
- Employee count by department and position
